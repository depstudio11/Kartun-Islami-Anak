# Kartun-Islami-Anak
Repository ini berisi halaman web sederhana untuk menampilkan Playlist Kartun Islami Anak dari DepStudio. Dibuat menggunakan HTML statis dengan pemutar YouTube bawaan, halaman ini ditujukan agar anak-anak dapat menonton konten islami secara mudah, aman, dan fokus tanpa gangguan dari elemen YouTube lain.
